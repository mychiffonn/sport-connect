## Outline User Roles

- Sport enthusiasts
- Game organizers

## Draft User Stories

1. As a sports enthusiast, I want to view all available games, so that I can choose one for my liking.
2. As a sports enthusiast, I want to be able to view a game's attendees, so that I can see who I'm playing with and against.
3. As a sports enthusiast, I want to be able to RSVP to a game, so that confirm my participation in game.
4. As a sports enthusiast, I want to be able to cancel an RSVP, so that I can leave at a given moment.
5. As a sports enthusiast, I want to be able to filter games by type of sports, locations, and events that are not full yet, so that I can search through attractive events.
6. As a game organizer, I want to be able to create a game, so that sports enthusiast can join to play.
7. As a game organizer, I want to be able to update a games details, so that all events are properly advertised.
8. As a game organizer, I want to be able to udpate the RSVP status, so that people can see whether or not they can participate.
9. As a game organizer, I want to be able to delete a game, so that, in case of emergencies, events can get canceled.
10. As a game organizer, I want to stop people from RSVP if my game reaches capacity, so that it doesn't exceed my event hosting capacity or venue capacity.
